FAIMMS_data_rss_channels_process
=============

This script downloads both qaqc and non qaqc data from the AIMS web service. The
data comes from the following sites :

* One Tree Island
* Orpheus Island
* Lizard Island
* Myrmidon Reef
* Davies Reef
* Rib Reef
* Heron Island
* Davies Reef

```bash
./faimms.py
```
